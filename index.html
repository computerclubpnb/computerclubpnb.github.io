Apa Itu Display ?

Display Block

Display Block merupakan properti CSS untuk membuat Layer Html mengalir ke samping dari Kiri ke Kanan

Display Inline

Display Inline merupakan properti CSS untuk membuat Layer Html Menutup Penuh Layar

Display Inline-Block

Display Inline Block merupakan properti CSS yang memiliki kriteria yang sama seperti display inline dan display block

Bagaimana Karakteristik dari masing masing dari properti berikut ?

Display Block

  • Mengalir ke Bawah
  • Lebarnya memenuhi Containernya
  • Dapat mengatur width dan height
  • Margin dan padding mempengaruhi elemen secara horizontal, maupun Vertikal

Display Inline

  • Mengalir ke Kesamping
  • Ukurannya sesuai dengan konten di dalamnya
  • Tidak dapat mengatur width dan height
  • Margin dan padding hanya mempengaruhi elemen secara horizontal, tidak Vertikal

Display Inline-block

  • Memiliki sifat dasar display inline (mengalir ke samping)
  • Dapat mengatur width dan height

Tag apa saja yang ada didalam properti berikut

Display Block

display inline terdiri dari tag div h1 - h6 hr p form ul, li , ol table dd,dl

Display Inline

berbeda dengan display block , display inline terdiri dari tag span a i b img label input textarea

Display Inline-block

display inline-block terdiri dari tag yang ada di Display Block dan Display Inline
